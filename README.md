# Chesse James

> A chess engine written in C++



## Features

* Representation: Bitboards
* Search
  * Negamax
  * Iterative Deepening
  * Alpha Beta
  * Quiscence search
  * Draw and Checkmate detection
* Move Ordering
  * MVV-LVA
  * Killer moves
  * History heuristic
  * Principal Variation
* Evaluation
  * Material evaluation
  * Bishop pairs
  * Rooks on open file
  * Position evaluation
* Transposition Table using Zobrist hashing
* Custom time management
